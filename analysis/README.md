## Analysis of The Office dialogue

Kotlin apps for dialogue analysis.

Database interactions using [Exposed](https://github.com/JetBrains/Exposed).

To build: 
```
./gradlew build
```

To build and install: 
```
./install.sh
```

To run correction and analysis:
```
./run.sh
```
